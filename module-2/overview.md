# Module 2: CSS Fundamentals - Base & Layout

## Overview

In Module 1, you built a complete 4-page website with semantic HTML structure. Your site works perfectly - all the content is there, all the links work - but it looks completely unstyled with browser default styling (black text on white background).

In this module, you'll transform your plain HTML into a **professionally styled dark-themed website** by implementing two CSS files: `base.css` and `layout.css`. You'll see a dramatic visual transformation as you add styles incrementally, learning fundamental CSS concepts along the way.

## What are Base & Layout CSS?

### base.css - Foundation Styles

`base.css` contains the **foundational styles** that apply across your entire website:

- CSS reset (consistent starting point)
- CSS custom properties (your design system variables)
- Typography (fonts, sizes, colors for text elements)
- Base element styles (links, lists, paragraphs)

Think of it as setting up your "design system" - the rules that make your site cohesive.

### layout.css - Structural Layout

`layout.css` contains styles for **structural components** that appear on every page:

- Container (centers and constrains content width)
- Header (navigation bar at the top)
- Footer (information at the bottom)
- Main content spacing
- Section organization

Think of it as the "skeleton" that holds your content together.

## What You'll Learn

- **CSS Reset** - Why and how to reset browser defaults
- **CSS Custom Properties (Variables)** - Modern way to manage design tokens
- **Box Model** - How CSS calculates element dimensions
- **Typography Hierarchy** - Creating visual hierarchy with font sizes and weights
- **Flexbox** - 1-dimensional layout for header and navigation
- **CSS Grid** - 2-dimensional layout for footer columns
- **Sticky Positioning** - Making the header stick to the top
- **Container Pattern** - Centering and constraining content width

## Why This Matters

### Foundation Before Details

By implementing base and layout CSS first, you establish a solid foundation. This approach:

- Creates consistency across all pages
- Makes adding page-specific styles easier later
- Teaches you the most impactful CSS first
- Follows professional development workflow

### Visual Transformation

This module provides the most dramatic visual change in the entire course. You'll go from completely unstyled HTML to a professional-looking dark-themed website, which is incredibly motivating!

### Modern CSS Practices

You'll learn CSS custom properties (variables), Flexbox, and Grid - the modern tools that professional developers use every day, not outdated techniques like floats or tables.
